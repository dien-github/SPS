## Hướng dẫn push code
- Ae push code tính năng mới vào nhánh mới nhen (đặt tên theo chức năng ví dụ 'feature/ir_task' - nếu như code khởi tạo đã check chắc chắn rằng build được và không lỗi vặt thì cứ push vào nhánh develop).
- Sau khi check không lỗi thì sẽ ae mới tạo merge request vào nhánh develop nhen. Điển merge vô cho.
- Khi nào các thành phần web, firmware, linux, docs của mình hoàn chỉnh rồi thì Điển sẽ merge vào nhánh main để làm Release.
